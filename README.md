### 📌 프로젝트 개요
이 프로젝트는 JWT(Json Web Token) 기반 인증 메커니즘을 통해 보안성을 강화하고, 역할(Role) 기반 접근 제어를 적용한 사용자 인증 시스템을 구축합니다.

### ✅ 주요 기능
- 회원가입 및 로그인 기능

- 일반 사용자(User) 및 관리자(Admin)의 회원가입/로그인 기능 제공

- JWT 기반 인증 처리

- 로그인 성공 시 Access Token 발급

- JWT 토큰을 통해 API 요청 인증 및 권한 검증

- 역할(Role) 기반 API 접근 제어

- 관리자(Admin) 권한이 필요한 API 보호

- 예: 관리자 권한 부여 API 등
